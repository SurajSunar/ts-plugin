### About Project

---

This is an experimental repo to understand the concept of building plugins for any sort of software.

### Example can be classified as:

---

- app: Start of the application
- Blog Engine: Main application which has features and expose provision to add plugins to enhance functionalities.
- Plugin Manager: Main task to register plugins and execute whenever needed.
- Plugins folder: New plugins can be developed and patched to the Blog engine( in this case).
